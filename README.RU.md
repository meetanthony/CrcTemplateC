# CRC Calculation Example in C

Этот репозиторий содержит пример реализации алгоритма расчёта CRC (циклический избыточный код) на языке C. Он состоит из двух файлов:

- `crc.c` — содержит универсальную функцию расчёта CRC поддерживающей настройки ширины, полинома, начального значения, XOR-выхода, а также флагов отражения входных и выходных данных;
- `main.c` — демонстрация использования функции `crc_compute` из `crc.c` на примере строки `"123456789"`.

## Параметры работы алгоритма

Параметры большинства известных алгоритмов CRC, включая полиномы, `init`, `xorOut`, `refIn`, `refOut`, можно найти на сайте:

[https://crccalc.com](https://crccalc.com)

## Параметры примера в данном репозитории: [CRC-32/ISO-HDLC](https://crccalc.com/?crc=123456789&method=CRC-32/ISO-HDLC&datatype=ascii&outtype=hex)

Ожидаемый результат CRC от строки `"123456789"`: `0xCBF43926`

## Компиляция

```bash
gcc -o crc_test main.c crc.c
./crc_test
```

## Ограничения

- Необходима заранее рассчитанная таблица CRC.
- Размер CRC до 32 бит.
- Не поддерживает различающиеся refIn и refOut (только если оба одновременно `true` или `false`).
